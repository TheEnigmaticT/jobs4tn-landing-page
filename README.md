# Jobs4TN + BigInterview Landing Page

A high-converting landing page for Tennessee job seekers, featuring:

- **Video Testimonials** - Mux-powered video carousel with smooth infinite scroll
- **Responsive Design** - Mobile-first, works on all devices
- **Fast Performance** - Single HTML file with inline CSS, no external dependencies except Google Fonts and Mux Player
- **Accessibility** - Semantic HTML, proper contrast, keyboard navigation

## Files

- `index.html` - Complete landing page with all CSS and JavaScript inline

## Dependencies (CDN)

- Google Fonts (Montserrat)
- Mux Player (@mux/mux-player)

## Deployment

This is a static HTML page that can be deployed anywhere:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting

Simply upload `index.html` to your hosting provider.
